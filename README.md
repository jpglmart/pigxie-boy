## Controls
### Keyboard
* Move - <kbd>A</kbd> / <kbd>D</kbd>
* Jump - <kbd>J</kbd>
* Attack - <kbd>K</kbd>
* Throw item - <kbd>L</kbd>
### Gamepad
* Move - <kbd>D-Pad</kbd>
* Jump - <kbd>A</kbd>
* Attack - <kbd>X</kbd>
* Throw item - <kbd>B</kbd>
